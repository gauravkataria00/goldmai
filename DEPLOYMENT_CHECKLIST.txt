🚀 GOLDMAI PRODUCTION DEPLOYMENT CHECKLIST

═══════════════════════════════════════════════════════════════

STEP 1: MONGODB DATABASE SETUP (10 minutes)
─────────────────────────────────────────────────────────────
□ Open: MONGODB_SETUP.txt (in this folder)
□ Follow all steps 1-10 to create MongoDB cluster
□ Get your Connection String
□ SAVE: MongoDB Connection String somewhere safe

⏱️ Time: ~10 minutes

═══════════════════════════════════════════════════════════════

STEP 2: RAILWAY BACKEND DEPLOYMENT (15 minutes)
─────────────────────────────────────────────────────────────
□ Open: RAILWAY_SETUP.txt (in this folder)
□ Follow all steps 1-6 to deploy backend
□ Add environment variables:
  ✓ MONGODB_URI = [Your MongoDB Connection String]
  ✓ JWT_SECRET = goldmai_secret_2024
  ✓ PORT = 5000
  ✓ NODE_ENV = production
□ Click "Redeploy"
□ Wait for deployment (5-10 minutes)
□ COPY: Your Railway URL: https://goldmai-api.railway.app

⏱️ Time: ~15 minutes

═══════════════════════════════════════════════════════════════

STEP 3: VERCEL FRONTEND DEPLOYMENT (5 minutes)
─────────────────────────────────────────────────────────────
□ Open: VERCEL_SETUP.txt (in this folder)
□ Follow all steps 1-9 to deploy frontend
□ Add environment variables:
  ✓ VITE_API_URL = https://goldmai-api.railway.app
□ Click "Deploy"
□ Wait for deployment (2-3 minutes)
□ COPY: Your Vercel URL: https://goldmai.vercel.app

⏱️ Time: ~5 minutes

═══════════════════════════════════════════════════════════════

STEP 4: CONNECT FRONTEND & BACKEND
─────────────────────────────────────────────────────────────
You don't need to do anything!

The environment variable VITE_API_URL already points your 
frontend to the backend API. Everything is auto-connected!

═══════════════════════════════════════════════════════════════

FINAL STEP: VERIFY EVERYTHING WORKS
─────────────────────────────────────────────────────────────
□ Open: https://goldmai.vercel.app
□ Check if homepage loads (you'll see big hero section)
□ Check if store cards appear with images
□ Check if animations work smoothly
□ Try clicking "Explore All 1000+ Stores" button

If all works ✅ → YOU'RE DONE! LIVE IN PRODUCTION! 🎉

═══════════════════════════════════════════════════════════════

TROUBLESHOOTING
─────────────────────────────────────────────────────────────

If homepage doesn't load or is blank:
1. Check Vercel deployment log for errors
2. Check if VITE_API_URL is set correctly

If API errors appear:
1. Check Railway deployment log
2. Verify MONGODB_URI is correct in Railway
3. Verify MongoDB has correct username/password

If database won't connect:
1. Check MongoDB connection string format
2. Verify IP address is whitelisted in MongoDB Atlas
3. Verify username and password are correct
